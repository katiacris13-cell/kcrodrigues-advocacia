# Site KC Rodrigues Advocacia

Projeto pronto para publicar na Vercel.

## Como publicar na Vercel

1. Baixe e descompacte este arquivo ZIP.
2. Entre em https://vercel.com.
3. Clique em **Add New > Project**.
4. Escolha a opção de importar/subir projeto.
5. Envie a pasta `kcrodrigues-advocacia`.
6. Na configuração do projeto, mantenha:
   - Framework Preset: Vite
   - Build Command: `npm run build`
   - Output Directory: `dist`
7. Clique em **Deploy**.

## Domínio

Depois do deploy:
1. Vá em **Settings > Domains**.
2. Adicione `kcrodrigues.adv.br`.
3. A Vercel mostrará os registros DNS necessários.
4. Entre no painel onde o domínio foi comprado e configure os DNS conforme indicado.
